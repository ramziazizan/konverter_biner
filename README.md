# Konverter Biner.Py

Proyek sederhana ini adalah kalkulator berbasis terminal yang dibangun menggunakan **Python**. Fungsinya adalah mengonversi bilangan biner (Basis 2) ke tiga sistem bilangan lainnya:

* Desimal (Basis 10)
* Oktal (Basis 8)
* Heksadesimal (Basis 16)
